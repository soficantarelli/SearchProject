Bienvenido a Cercare!

Cercare es una plataforma utilizada para empresas para disponibilizar la obtención de resultados de búsquedas a través de sitios web y servicios proporcionados.

Solicita la creación de tu usuario y comienza a disfrutar de las funcionalidades disponibles

Funcionalidades

  1. Agregado de servicios con sus sitios web
  2. Agregado de sitios web
  3. Indexación de metadatos
  4. Buscador personalizable
  5. Búsqueda por voz a texto 
  6. Tablero de estadísticas
